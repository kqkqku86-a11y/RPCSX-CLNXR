# RPCSX-UI-Android — Ouroboros (experimental)

A native Android front-end for **PlayStation 3** emulation on arm64. A
continuously-updated fork of
[RPCSX/rpcsx-ui-android](https://github.com/RPCSX/rpcsx-ui-android). The actual
emulation is done by the separate
[rpcsx core](https://github.com/Ouroboros420/rpcsx), loaded at runtime.

### Honest note on how it's made
Updates here — and especially the work to merge newer
[RPCS3](https://github.com/RPCS3/rpcs3) changes into the core — are done with
heavy **AI assistance** (Claude). Treat it as **experimental**: verify before
relying on it, and expect rough edges.

### Why this exists
Not to create yet another permanent fork. The goals are simply to **stay
current**, and to **help the real RPCSX / RPCS3 teams** by demonstrating what
can be done — they are welcome to take anything useful from here. All credit for
the emulator and this UI belongs to the **RPCS3** and **RPCSX** developers.

> Piracy is not permitted. Do not ask for games or system files.
